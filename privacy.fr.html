Politique de confidentialité

L'application « Palette Store » (l'« App »)

Le développeur de l'App (le « Développeur », « nous ») respecte votre vie privée et s'engage à protéger vos informations personnelles. Cette politique explique comment les informations sont traitées dans l'App.

1. Informations que nous collectons

L'App ne collecte aucune information permettant de vous identifier personnellement (nom, e-mail, numéro de téléphone, localisation, contacts, etc.).

Les informations traitées par l'App sont les suivantes :

  • Scores, progression et réglages du jeu : enregistrés localement sur votre appareil. Si iCloud est activé, ils sont synchronisés via iCloud d'Apple (stockage clé-valeur) et partagés entre les appareils utilisant le même compte Apple.
  • Données Game Center : lors de l'utilisation des classements ou des succès, elles sont enregistrées via Game Center d'Apple.

2. Partage avec des tiers et transmission externe

  • Nous n'envoyons ni ne stockons vos informations sur un serveur exploité par nous.
  • L'App n'affiche aucune publicité. Nous n'utilisons aucun réseau publicitaire tiers ni SDK d'analyse (suivi).
  • Nous n'utilisons pas vos données à des fins de suivi.

3. Services Apple

L'App utilise iCloud et Game Center, des services fournis par Apple. Les informations traitées via ces services sont soumises à la politique de confidentialité d'Apple. Pour plus de détails, consultez la Politique de confidentialité d'Apple.

4. Utilisation par les enfants

L'App est conçue pour que les enfants puissent l'utiliser en toute sécurité : elle ne collecte pas d'informations personnelles et n'affiche pas de publicité ciblée comportementale. À utiliser sous la surveillance d'un parent ou tuteur.

5. Suppression des données

Les données enregistrées sur votre appareil sont effacées lorsque vous supprimez l'App. Les données synchronisées avec iCloud peuvent être gérées et supprimées depuis votre appareil dans Réglages > Compte Apple > iCloud. Les données Game Center peuvent être consultées dans les réglages de Game Center.

6. Modifications de cette politique

Nous pouvons réviser cette politique si nécessaire. En cas de changements importants, nous les annoncerons sur cette page.
